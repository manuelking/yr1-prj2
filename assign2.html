About this project

Assignment: Introduction to Web Programming - Assignment 2


Programming anguages and tools: HTML, CSS, JavaScript.

About: We were required to build a 'Phone Contract Finder' application that had to be developed using HTML, CSS and JavaScript. It would allow users to enter preferences for the type of phone they would like (brand, monthly cost, minutes etc.) and, based on these preferences, the application would recommend suitable phones and plans (phone contracts).

How to use:
1. Enter a brand
2. Enter a minimum data allowance
3. Enter a minimum number of talk minutes
4. Enter a minimum text allowance
5. Enter a minimum monthly cost
6. Observe the plan(s) that match your preferences
